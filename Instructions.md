## Requirements
- Python 3.8+
- PyTorch
- Transformers
- Torchaudio
- Matplotlib

## Pretrained Model Setup
This project uses the **Wav2Vec2-Base** pretrained model.  
Please download it from Hugging Face and place it in a local folder before running.
